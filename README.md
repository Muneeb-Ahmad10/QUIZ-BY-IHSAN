# QUIZ-BY-IHSAN
Simple C++ Quiz Program that asks questions and shows score

## How to Compile & Run
1. Open terminal/command prompt in this folder
2. Compile: g++ QuizPro.cpp -o quiz
3. Run: ./quiz   // For Windows use: quiz.exe

## About
Made using classes in C++. Asks 10 MCQs and calculates final score.
